# Security Policy

## 支持版本

| Version | Supported |
| --- | --- |
| v1.0.x | Yes |

## 报告安全问题

如果你发现安全问题，请优先通过 GitHub Security Advisory 或私下联系仓库维护者，不要在公开 Issue 中直接贴出可利用细节。

报告时请包含：

- 受影响版本
- 复现步骤
- 影响范围
- 相关日志或截图
- 你建议的缓解方式

## 本项目的安全关注点

- 用户配置只写入 `%AppData%\YueXinMiaoPet\config.json`，不写入安装目录。
- 日志写入 `%AppData%\YueXinMiaoPet\logs\app.log`，用于本地排查。
- 程序不会上传用户本地数据。
- 自定义 GIF 目录只读取本地 `.gif` 文件。
- 天气服务仅请求公开天气接口需要的经纬度信息。
- 天气接口失败、断网或返回异常时会安全回退，不应导致崩溃。
- 安装脚本只安装程序文件、资源文件和必要快捷方式，避免误删用户文件。
- 卸载流程不应删除用户自定义 GIF 目录。

## 不在范围内

- 用户自行下载或放入的第三方 GIF 内容版权与安全性。
- 被用户手动替换的安装脚本或非官方安装包。
- 非官方分发渠道中的安装器篡改。
