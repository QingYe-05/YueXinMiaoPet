# 更新日志

本项目遵循简化版语义化版本记录：`主版本.次版本.修订版本`。

## [1.0.0] - 2026-06-26

### 新增

- 基于 C# / WPF / .NET Framework 4.8 的 Windows 桌宠应用。
- 支持 Windows 7 SP1、Windows 10、Windows 11。
- 透明、无边框、可拖动、可置顶的桌宠窗口。
- 内置 13 类月薪喵 GIF 分类资源，共 186 张 GIF。
- 支持中文路径和中文 GIF 文件名。
- 支持心情卡片选择、心情有效期和心情分类强绑定。
- 支持单击桌宠时只在当前心情分类内切换 GIF。
- 支持自定义 GIF 目录，并可一键切回内置月薪喵分类资源。
- 支持天气服务、省市选择、天气提示气泡和 10 分钟自动刷新。
- 支持缩放比例和透明度 Slider 调节。
- 支持托盘菜单、设置窗口、资源调试面板。
- 支持 Inno Setup 安装脚本和 .NET Framework 4.8 Runtime 检测。

### 修复

- 修复点击桌宠后从当前心情跳回普通 GIF 的问题。
- 修复 GIF 资源扫描可能重复加载的问题。
- 修复心情保存后不立即生效的问题。
- 修复自定义 GIF 目录无效时没有安全回退的问题。

### 已知限制

- Windows 7 需要 SP1 和 .NET Framework 4.8。
- 天气服务依赖网络，断网时会使用缓存或显示 unknown。
- GIF 分类依赖资源目录和 `mood_category_map.json`，新增资源后建议重新扫描。

- 改进 GIF 分类和标签推断规则。
